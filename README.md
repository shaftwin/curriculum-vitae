## Installation

```zsh
# Clone this repository
$ git clone https://github.com/shaftwin/curriculum-vitae.git

# Go into the repository
$ cd curriculum-vitae

# Install dependencies
$ yarn

# Run the app
$ yarn start
```

## Tests

```zsh
# Run the tests
$ yarn test
```

## Translation
<br>

> For now only en / fr are translated
<br>

```zsh
# Run the translation scanner
$ yarn local:extract
```

## Storybook

```zsh
# Run the tests
$ yarn storybook
```
